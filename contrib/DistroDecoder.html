Linux Distribution Decoder

Introduction

All the major Linux distributions draw from the same FOSS wells. They do not all draw the same things. This guide is intended to help administrators translate between distros.

Bare Essentials

For your convenience here are some of the essential commands you will need.

    Managing Services with systemd

    A new system startup daemon, systemd, replaces the startup scripts that we have loved for decades. We will use

    • To set the docker service to start at boot: 
      systemctl enable docker
    • To set the docker service to start at immediately: 
      systemctl start docker

    Firewalld

    • To permanently open port 80/tcp: 
      firewall-cmd --permanent --add-port=80/tcp
firewall-cmd --reload
    • To list open ports: 
      firewall-cmd --list-all

    UFW

    • To permanently open port 80/tcp: 
      ufw allow 80/tcp
    • To list open ports: 
      ufw status
