Part B: Users and Groups (3 points)

In this section you will add two new users to s01, andy and amita. Andy and Amita are both web site admins.

  1. Create two new users on s01 with user names of: andy and amita
  2. Give both new users the initial passwd of mohawk1
  3. Force each user to change their passwd the next time they log on.
  4. Create a group named web containing andy and amita

Part C: Add 2 disks (3 points)

  1. Add two 2GB disks to s01
  2. Use the disks you just added to create a new LVM volume group named vgWeb
  3. Create a single logical volume named /dev/vgWeb/lvol0 using all the space available in vgWeb
  4. Build an ext4 file system on /dev/vgWeb/lvol0
  5. Mount the new file system at /mnt/web, this mount must occur autamitically when s01 is rebooted

Part D: Share /mnt/web (2 points)

  1. Configure /mnt/web to be used by the web group to share files.

Part E: Install The Apache Web Server (3 points)

Andy and Amita need the Apache web server to be running.

  1. Install the web server
  2. Configure the web server to start automatically at boot
  3. Make sure that the web server is running (start it manually or reboot)

Part F: Welcome Page (2 points)

Configure s01 such that the default web page contains the message, Amita and Andy. Formatting is not important.

Note: The grading script will use wget to check this. Make sure wget is installed an working. wget-1.19.5-10.el8.x86_64.rpm

Part G: Apache Gets Nights Off (2 points)

To encourage users to get a good night's sleep, it has been decided that the web server will be unavailable from 23:30 each night until 07:00 the next morning.

Create a cron file named /etc/cron.d/web that stops httpd each evening at 23:30 and restarts it the next morning at 07:00.

Run the grading script host_info_t1.sh and submit the output to Canvas.
