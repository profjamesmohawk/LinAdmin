COMP-10018: Practice Test 1

This practice test will help you prepare for Test 1. Test 1 will be very similar to this test.

Prerequisites:

Before you start this lab you must have:

  • The comp-10018 virtual lab appliance, comp-10018-vlab.ova found on the CSAIT lab systems.
  • Oracle VirtualBox software, found on all lab systems at the college and available from http://virtualbox.org

Part A: Import The Appliance

  1. Import clean, fresh copies of w01 and s01 from (D:/Courses/comp-10018/comp-10018-vlab.ova)

Part B: Users and Groups (3 points)

In this section you will add two new users to s01, sally and sue. Sally and Sue are twins, and as such like to share files.

  1. Create two new users on s01 with user names of: sally and sue
  2. Give both new users the initial passwd of mohawk1
  3. Force each user to change their passwd the next time they log on.
  4. Create a group named twins containing sally and sue

Part C: Add a disk (3 points)

  1. Add a 3GB disk to s01
  2. Use the entire disk you just added to create a new LVM volume group named vg01
  3. Create a 2GB logical volume named /dev/vg01/lvol0
  4. Build an ext4 file system on /dev/vg01/lvol0
  5. Mount the new file system at /twins, this mount must occur autamitically when s01 is rebooted
  6. sue and sally must have full access to /twins, other regular users must have no access

Part D: Install mysql (3 points)

Sally and Sue are database uses and they need the myslq database installed and running.

  1. Install the mysql database server (hint: mysql-server)
  2. Configure the mysql database server to start automatically at boot (hint: mysqld)
  3. Make sure that the mysql server is running (start it manually or reboot)

Part E: Twins share everything (1 point)

Sue and Sally want to share as many of their files as possible, even the files in their home directories.

  1. Configure their home directories so that each has read access to the other's home directory without giving other regular users any access
  2. Configure their accounts so that by default every file they create has a group of twins (hint: initial login group)
