COMP-10018: Test 1 (Make Up)

Prerequisites:

Before you start this lab you must have:

  • The comp-10018 virtual lab appliance, comp-10018-vlab.ova found on the CSAIT lab systems.
  • Oracle VirtualBox software, found on all lab systems at the college and available from http://virtualbox.org

Grading:

You will run a host info script on s01 and submit the output to eLearn.

Time:

You have 60 minutes to complete the exam. Any additional time in the exam period may be used for running the grading script and uploading to eLearn.

Part A: Import The Appliance

  1. Import clean, fresh copies of w01 and s01 from (D:/Courses/comp-10018/comp-10018-vlab.ova)

Part B: Users and Groups (3 points)

In this section you will add two new users to s01, andy and amita. Andy and Amita are both web site admins.

  1. Create two new users on s01 with user names of: andy and amita
  2. Give both new users the initial passwd of mohawk1
  3. Force each user to change their passwd the next time they log on.
  4. Create a group named webadmin containing andy and amita

Part C: Add 3 disks (3 points)

  1. Add three 1GB disks to s01
  2. Use the disks you just added to create a new LVM volume group named vgWeb
  3. Create a single logical volume named /dev/vgWeb/lvol0 using all the space available in vgWeb
  4. Build an ext4 file system on /dev/vgWeb/lvol0
  5. Mount the new file system at /webadmin, this mount must occur autamitically when s01 is rebooted

Part D: Install The Apache Web Server (3 points)

Andy and Amita need the Apache web server to be running.

  1. Install the web server
  2. Configure the web server to start automatically at boot
  3. Make sure that the web server is running (start it manually or reboot)

Part E: Welcome Page (2 points)

Configure s01 such that the default web page contains the message, Amita and Andy. Formatting is not important.

Note: The grading script will use wget to check this. Make sure wget is installed an working.

Part F: Apache Gets Weekends Off (2 points)

It has been decided that the web site will not be available on weekends.

Create a cron file named /etc/cron.d/web that stops httpd each Friday evening at 20:15 and restarts it the next Sunday morning at 06:00.

Part H: Grading

Run the grading script host_info_t1.sh and submit the output to eLearn.
