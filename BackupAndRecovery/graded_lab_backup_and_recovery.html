Graded Lab: Backup And Recovery

Introduction

Today you will use the tools from the "Backup And Recovery" lab to archive data and to migrate data.

Prerequisites:

Before you start this lab you must have:

  • The comp-10018 virtual lab appliance, comp-10018-vlab.ova found on the CSAIT lab systems.
  • Oracle VirtualBox software, found on all lab systems at the college and available from http://virtualbox.org

Grading:

You will run a host info script on s01 and submit the output to eLearn.

Note: All tasks will be performed on s01 unless otherwise noted.

Part A: Simple tar Archives

  1. Create a tar archive named /tmp/books.tar containing the contents of /opt/Mohawk/books
  2. Create a gzip compressed tar archive named /tmp/books.tgz containing the contents of /opt/Mohawk/books

Part B: Simple cpio Archives

  1. Create a cpio archive named /tmp/books.cpio containing the contents of /opt/Mohawk/books (make sure the archive uses relative paths)
  2. Create a gzip compressed cpio archive named /tmp/books.cpio.gz containing the contents of /opt/Mohawk/books (make sure the archive uses relative paths)

Part C: Incremental Archives with tar

For this section you will create an initial, "level 0", archive and one incremental, "level 1", archive. Use tar's built in support for incremental archives.

  1. Store your archive set in /tmp/opt_set
  2. Name the initial backup /tmp/opt_set/level_0.tar and have it contain the contents of /opt
  3. Create a new "book" in /opt/Mohawk/books
  4. Create an incremental backup named /tmp/opt_set/level_1.tar

Part D: Migrating Data

Our backup and recovery tools can also be used to migrate data. For this exercise we will create a copy of /home/alice from w01 to s01

  1. Copy the contents of /home/alice from w01 to a /home/alice on s01 preserving all meta-data. You may use any commands you like including: cpio, scp, tar, rsync, ssh
  2. Write a short description of how you accomplished this task in a text file on s01 named /tmp/migrate.txt

Part E: Migrating Data - Puzzle

Using only tar and ssh to complete "Part D" above using a single command pipe-line run as root from w01. Write this command pipe-line to a text file on s01 named /tmp/puzzle.txt
