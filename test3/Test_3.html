Grading:

You will run a host info script on s01 and submit the report to Canvas.

  • host_info_t3.zip

Part A: Build this configuration (10 points)

  1. Create s03 and r01 by cloning s01
  2. Update /etc/hosts files on s01
  3. s01 and s03 must be able to communicate via IP
  4. Remember to stop and disable the firewalld service on r01.
  5. Make no changes to w01 (w01 does not need to ping s03)

Part B: Backups (3 points)

  1. On s01 create a cpio archive named /tmp/conf.cpio containing every file residing in the /etc directory tree who's name ends in .conf
  2. On s01 create a tar archive named /tmp/root_home.tar containing every file residing in root's home directory.

Part C: ssh (2 points)

  1. Write a single command line that will create a file named /tmp/1.txt on s03 containing the text "hello".
  2. Your command line must start with: echo hello | ssh root@s03
  3. Test your command by running it on s01
  4. Create a text file named /tmp/ssh_cmd.txt on s01 containing your command line.
